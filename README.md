# 🎵 YouTube → MP3 Downloader

A clean, fast YouTube to MP3 converter built with **Node.js + Express + ytdl-core**.

![Preview](https://i.imgur.com/placeholder.png)

---

## 🚀 Run Locally

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/yt-mp3-downloader.git
cd yt-mp3-downloader
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the server
```bash
npm start
```

### 4. Open in browser
```
http://localhost:3000
```

---

## 📦 Tech Stack

| Layer     | Technology                  |
|-----------|-----------------------------|
| Backend   | Node.js + Express           |
| Downloader| ytdl-core                   |
| Converter | fluent-ffmpeg + ffmpeg      |
| Frontend  | HTML + CSS + Vanilla JS     |

## 🌐 Deploy to Render (Free)

1. Push this repo to GitHub
2. Go to [render.com](https://render.com) → New Web Service
3. Connect your GitHub repo
4. Set:
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
5. Deploy! ✅

> ⚠️ **Note:** GitHub Pages is static-only and **cannot** run this Node.js server.
> Use [Render](https://render.com), [Railway](https://railway.app), or [Fly.io](https://fly.io) for free hosting.
